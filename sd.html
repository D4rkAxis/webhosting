Connect to your GitHub Gist database

GIST CONNECTION

RAW GIST URL

This is the URL the site fetches every 5 minutes. No auth needed for a secret Gist if you know the URL.

Enter your Gist ID above to see the URL.

HOW IT WORKS

1

Create a GitHub Gist

Go to gist.github.com → New Gist → filename tickets.json → content [] → Secret → Create. Copy the Gist ID from the URL.

2

Create a GitHub Token

GitHub → Settings → Developer Settings → Tokens (Classic) → tick gist scope → Generate. Paste above — stored in session only.

3

Create a private GitHub repo for the sync worker

Create any private repo. Under Settings → Secrets → Actions add: BITRIX_WEBHOOK, GIST_TOKEN, GIST_ID.

4

Add the two sync files to the repo

Put sync.js at scripts/sync.js and the workflow at .github/workflows/sync-bitrix-to-gist.yml. Push → Actions runs every 10 min.

5

Enter config above and click Refresh

Paste Gist ID + token, click Save & Test. The hub auto-refreshes every 5 minutes.
